O que é HTML?

HTML é a sigla para HyperText Markup Language, que em português significa Linguagem de Marcação de Hipertexto. Ela é usada para criar e estruturar páginas da internet. Com o HTML, você pode definir títulos, parágrafos, links, imagens, listas e muitos outros elementos que compõem um site. O HTML não é uma linguagem de programação, porque ele não executa ações ou comandos lógicos como "se" ou "enquanto". Em vez disso, ele apenas organiza o conteúdo da página, dizendo o que é um título, o que é um parágrafo, o que é um link, etc. Toda página da web que você acessa foi construída com HTML (junto com outras tecnologias como CSS e JavaScript). Por exemplo, quando você vê um título grande em um site, uma imagem ou um link clicável, o HTML está por trás disso, definindo como esse conteúdo deve aparecer e se comportar.

Importância

O HTML é apenas uma parte da construção de uma página web. Ele define a estrutura e o conteúdo do site, mas trabalha em conjunto com outras tecnologias para criar experiências completas e modernas.

  • O CSS (Cascading Style Sheets) é usado junto com o HTML para definir o estilo visual da página: cores, fontes, tamanhos, espaçamentos, posicionamento e responsividade.
  • O JavaScript entra para adicionar interatividade e lógica ao site, como formulários dinâmicos, menus interativos, animações, validações e muito mais.

Além disso, existem ferramentas e bibliotecas modernas, como React, Vue.js e Angular, que combinam HTML, CSS e JavaScript para facilitar o desenvolvimento de interfaces mais rápidas, escaláveis e organizadas. Compreender como o HTML se encaixa nesse ecossistema é essencial para qualquer desenvolvedor web, pois é a base sobre a qual tudo o mais é construído. Para um desenvolvedor web, o HTML é essencial, pois é o ponto de partida de qualquer site ou aplicação web. Mesmo que existam outras tecnologias como CSS (para aparência) e JavaScript (para comportamento), tudo começa com o HTML. Para qualquer desenvolvedor web, conhecer HTML é indispensável. É a linguagem que dá forma ao conteúdo e serve como base para as demais tecnologias da web. Sem HTML, simplesmente não existe página web funcional.

Tags

As tags são os blocos de construção do HTML. Elas indicam qual tipo de conteúdo você está colocando na página (como título, parágrafo, imagem, link etc.). A maioria das tags tem uma abertura <tag> e um fechamento </tag>.

Principais TAGS:

  • <h1> a <h6>: São títulos ou cabeçalhos, indo do maior (<h1>) ao menor (<h6>).
  • <p>: Cria um parágrafo de texto.
  • <a> Cria um link para outra página ou site.
  • <ol>: Cria uma lista ordenada (com números).
  • <ul>: Cria uma lista não ordenada (com bolinhas).
  • <li>: Cada item dentro de uma lista.
  • <strong>: Deixa o texto em negrito e com ênfase.
  • <i>: Deixa o texto em itálico.
  • <u>: Deixa o texto sublinhado.
  • <div>: organiza e separa o conteúdo da página.
  • <mark>: Destaca um trecho do texto, como este exemplo marcado.
  • <small>: Deixa o texto menor, como este exemplo pequeno.
  • <hr>: Insere uma linha horizontal para separar conteúdos.
  • <sub>: Texto em subscrito (abaixo da linha), útil para fórmulas como H2O.
  • <sup>: Texto em sobrescrito (acima da linha), útil para potências como x2.
  • <blockquote>: Cita um trecho de outro autor ou fonte. Exemplo:
    • "O HTML é a espinha dorsal de qualquer página da web." – Desenvolvedor Anônimo
  • <font>: Alterna a fonte, tamanho e a cor do texto - MAS ESTÁ EM DESUSO
  • <span>: Alterna a fonte, tamanho e a cor do texto.

Listas

As listas em HTML são usadas para organizar informações de forma estruturada. Elas são muito úteis para exibir passos, itens, tópicos ou qualquer grupo de elementos relacionados.

Tipos de listas:

  • <ul>: Lista não ordenada (unordered list) – usa marcadores (bolinhas).
  • <ol>: Lista ordenada (ordered list) – usa números.
  • <li>: Cada item da lista (list item) – usado dentro de <ul> ou <ol>.
